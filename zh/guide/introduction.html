# Live777

# 概述

  Live777是一个开源的（MIT协议）简单高效的实时视频服务器，支持WHIP/WHEP协议。
  Live777媒体服务器和Gstreamer (opens new window)、FFmpeg (opens new window)、OBS Studio (opens new window)、VLC (opens new window)、 WebRTC (opens new window)等客户端配合使用，提供流的接收和分发的能力，是一个典型的发布 （推流）和订阅（播放）服务器模型。 Live777支持互联网广泛应用的音视频协议转换，比如可以将RTP， 转成WHIP或WHEP等协议。
  Live777使用rust开发，可以在Linux、Windows、macOS等多个平台运行， 我们提供的镜像也是基于Windows 10构建的。

# 特性

# 支持WHIP/WHEP

# - SFU架构

  本服务器是一款基于SFU架构的中心节点媒体服务器，只负责转发，不做合流、转码等资源开销较大的媒体处理工作，将编码解码等工作分别放在发送端和接收端，因此大大降低了服务器的压力，放宽了对服务器配置的严格要求。
