# Build

You can build Live777 from source code, but docker is highly recommend.

# Live Streaming

Live777 supports live streaming.

# Prerequisites

  • Install rust(Download (opens new window)).
  • Terminal for accessing VitePress via its command line interface (CLI).

Note:The project is based on rust, So please install rust in advance

# Run Live777

  • Get Live777 source:
git clone https://github.com/binbat/live777.git
  • Build Live777 in /Live777/:
cd live777
cargo build
  • Run Live777 server:
cargo run

# Publish stream

  • publish stream by Gstreamer (opens new window) or OBS :

Note:OBS Studio version 30 or higher.
Note:OBS WHIP Current only support H264 video codecs and Opus audio codecs.

gst-launch-1.0 videotestsrc ! videoconvert ! vp8enc ! rtpvp8pay ! whipsink whip-endpoint="http://localhost:3000/whip/777"

# Play stream

  • open your browser,enter the URL:http://localhost:3000/
